Found a bug?

If you have found a bug, please send us an email (above) remember to tell us how to recreate the bug, browser you using, steps taken to create the bug, how long you been using your account, and other information that might help us. Alternatively if you have a github account, you can make an issue on our public git repo website https://github.com/panjiw/Team-Player/issues?labels=bug&state=open and we will response asap.
