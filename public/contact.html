Found a bug?

We have a public list of bugs and issues.

If you have a bug to report, you can either use the above link and post an issue with a github account or you can email us.
Remember to:

  1. Tell us what browser you are using, and how long you've had a TeamPlayer account.
  2. Give us some information about the bug. It must be clear, such as "clicking on home takes me to this page instead of that page," not "the home button doesn't work." List the steps we need to reporduce this bug. What did you do to reach this problem? What were you tryingto do?
